	via shortcodes or page builder elements and gutenberg blocks.

== Description ==

Almost everything is ready to create combo blocks from any post types, with few click you can generate beautiful grid for your blog post, product showcase, team member showcase, portfolio, gallery, archive post display, category post display, tags post display and custom taxonomy and terms post can be displayed via combo blocks. if you have basic knowledge in CSS you can style your own via layout editor to create unique style of your grid.

**Responsive Grid**
Combo Blocks is responsive and almost work in any device. good things is you can set some option based on mobile or tablet
device. like you can set custom column number on mobile device.

**Any Custom Post**
Combo blocks is master of displaying grid from any custom post types. You will never ever worried about. you can also
select multiple post types to display on single combo blocks.

**Taxonomy & Terms Support**
Query post by custom taxonomy and terms is the most wanted feature already in combo blocks, you can select multiple taxonomy and terms for query post.

**Unlimited Combo Blocks**
There is no limitation of creating combo blocks, you can create unlimited combo blocks on your website.

**Skin & Layout Based**
Combo blocks items based on skin and layout based, you can customize layout elements and change skin, if you have basic CSS
 knowledge you can customize layout elements and apply your own style.

**Pagination Support**
Combo Blocks has pagination support, thats mean if you combo blocks has many post so user can go through each pagination to
see your post archive. you can limit the pagination item count and "Next" & "Previous" text as well, you can also use
icons instead of text for these.

**Search Input Field**
you can display search input field at top of grid, so user can search though your grid and find their desired post.

**Post Query Features**
Combo Blocks has many support and input to display your desired post on the grid, you can customize

* Post types,
* Post categories, tags
* Taxonomies & terms, taxonomy relation, terms relation,
* Post publish status,
* Post order and
* Order by, order by meta field value,
* Custom number of posts per page, offset
* Exclude post by ids, and
* Display combo blocks based on keyword search.


**Layout settings**
You can customize grid item layout settings, following option avilable to customize

* Grid item width, you can set custom width for grid item based on mobile, tablet and desktop device. you can also set %
 based column, like 30% will gives you 3 column in each row.
* Grid item height - you can set custom height of grid item, based on device you can set auto height, fixed height or 
max height.
* Grid item background color.
* Grid item padding.
* Grid item margins.
* Lazy load - you can enable lazy load to hide grid on page load.
* Featured image custom size selection.
* Custom media source - there is 3 different source available in free version. Featured Image, First images from
content,
 Empty thumbnail
* Masonry style grid also available.

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Combo Blocks</strong>" activate it.<br />

After activate plugin you will see "Combo Blocks" menu at left side on WordPress dashboard click "New Combo Blocks" and use the options field "Combo Blocks"<br />

<br />
<strong>How to use on page or post</strong><br />
When Combo Blocks options setup done please publish Combo Blocks as like post or page<br />

and then copy shortcode from top of <strong>Combo Blocks Options</strong> `[post_grid  id="1234" ]`<br />

then paste this shortcode anywhere in your page to display grid<br />
